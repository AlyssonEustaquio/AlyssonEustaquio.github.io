1. Elementos Semânticos

Exemplo de uso de elementos semânticos para estruturação de conteúdo.

Artigo de Exemplo

Publicado em 14 de Julho, 2025

Este é um artigo fictício demonstrando elementos semânticos como article, header, e time.

Autor: Alysson

<article class="semantic-example">
    <header>
        <h3>Artigo de Exemplo</h3>
        <p>Publicado em <time datetime="2025-07-14">14 de Julho, 2025</time></p>
    </header>
    <p>Este é um artigo fictício...</p>
    <footer>
        <p>Autor: Alysson</p>
    </footer>
</article>

2. Formulário Interativo

Um formulário com diferentes tipos de entrada para interação do usuário.

<form class="form-example">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" placeholder="Digite seu nome" required>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" placeholder="Digite seu email" required>
    <label for="tipo">Tipo de Usuário:</label>
    <select id="tipo" name="tipo">
        <option value="estudante">Estudante</option>
        <option value="profissional">Profissional</option>
    </select>
    <button type="submit">Enviar</button>
</form>

3. Tabela de Dados

Uma tabela estilizada para exibir dados organizados.

Nome Idade Profissão
Alysson 25 Aprendiz
Afonso 22 Desenvolvedor 
<table class="data-table">
    <thead>
        <tr>
            <th>Nome</th>
            <th>Idade</th>
            <th>Profissão</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Alysson</td>
            <td>25</td>
            <td>Aprendiz</td>
        </tr>
        <tr>
            <td>Afonso</td>
            <td>22</td>
            <td>Desenvolvedor</td>
        </tr>
    </tbody>
</table>
