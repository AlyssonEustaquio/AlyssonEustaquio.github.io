HTML - Linguagem de Marcação

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ele descreve a estrutura de uma página web usando elementos.

Exemplo Básico de HTML

<!DOCTYPE html>
<html>
<head>
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Bem-vindo ao HTML!</h1>
    <p>Este é um parágrafo de exemplo.</p>
    <a href="https://www.w3schools.com">Visite o W3Schools</a>
</body>
</html>

CSS - Folhas de Estilo

CSS (Cascading Style Sheets) é a linguagem usada para estilizar documentos HTML. Ele controla a aparência e o layout dos elementos.

Exemplo de CSS

body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
}

h1 {
    color: white;
    text-align: center;
    background-color: #333;
    padding: 20px;
}

p {
    font-size: 18px;
    color: #333;
    margin: 15px 0;
}

JavaScript - Programação Web

JavaScript é a linguagem de programação da web. Permite adicionar interatividade e comportamento dinâmico às páginas.

Exemplo Básico de JavaScript

// Variáveis
let nome = "João";
const idade = 25;

// Função
function saudar() {
    return "Olá, " + nome + "! Você tem " + idade + " anos.";
}

// Manipulação do DOM
document.getElementById("resultado").innerHTML = saudar();
